# ADR-0001：選擇題計時的時間模型

- **狀態**：暫定（Provisional）—— 已採用為第一版實作，但**尚未定論、開放修正**。
- **日期**：2026-06-24
- **決策者**：專案維護者
- **相關**：計時功能（P2）、歷史紀錄耗時欄（未來校準資料來源）

## 背景（Context）

考選部國家考試的科目分兩種題型：

- **純測驗式**（只有選擇題）：護理師（每科 60 分）、律師第一試綜合法學（公/刑/民/商，90/90/100/80 分）。
- **申論式與測驗式混合式**（同一張卷、同一時段，選擇題＋申論題一起作答）：社工師、諮商心理師、臨床心理師（各 120 分）、會計師（中會/審計 180 分、稅務 120 分）。

本平台目前把**選擇題與申論分開練**（選擇題在整卷/模擬，申論在申論分頁）。要為「考試形式」模式加上倒數計時以擬真，但混合式科目的官方時間**含申論**，若選擇題整卷直接套官方全時間會過度給時。

實證觀察：純測驗科目「官方時間 ÷ 官方題數」皆 ≈ **1.2 分/題**（60÷50、90÷75、100÷80、80÷70…）。此係數可作為「選擇題應得時間」的基準。

## 決策（Decision，暫定）

1. **碼錶（count-up）全面移除，一律改倒數。**
2. **完整版模式（新增）**：選擇題＋申論合併、同一時段作答，倒數＝**官方完整時間**（如社工 120 分），到 0 強制交卷。此為混合式科目的真擬真；純測驗科目無申論，即整卷用官方全時間。
3. **單獨選擇題模式（兩段式倒數，不強制中斷）**：
   - 倒數總長＝**官方完整時間**（如 120 分），可用滿。
   - 在「選擇題建議時間＝題數 × 1.2 分」（如 40 題 ≈ 48 分）標記點之後，**繼續倒數的數字轉為紅色**，提示已超過選擇題應有節奏、正在吃進原申論時段。
   - **不在 48 分強制交卷**（善待練習者，不因網路斷線/誤關/思考久而被硬打斷）；僅在全時間 0 時強制交卷。
   - 畫面同時清楚標注：「選擇題建議 48 分・全科完整 120 分（含申論）」。
   - 純測驗科目：建議時間 ≈ 全科時間，兩段重合，等於單一倒數。

## 考慮過的替代方案（Alternatives）

- **A. 混合式選擇題用官方全時間為硬底線**：簡單，但選擇題練習被給了含申論的全部時間，**無法練到選擇題應有的速度感**（偏鬆）。
- **B. 以 1.2 分/題為硬底線、到點強制交卷**：嚴格擬真選擇題節奏，但**對仍在學習者可能太硬**，且抹掉「全科時間感」（你不知道真考場一科共多久）。
- **採用＝A/B 折衷**：全時間可用（不硬逼）＋ 48 分後轉紅（保留全科時間感，同時提醒選擇題節奏）。兼顧「擬真」與「善待練習者」。

## 後果（Consequences）

- 練習者不會因超過選擇題建議時間就被打斷，符合「防誤關/斷線、善待真正想練的人」的設計原則。
- 紅色提示傳達擬真節奏，但不具強制力——對「想嚴格逼自己」的人可能不夠。
- **1.2 係數是經驗值**，且混合式科目的「選擇題應得時間」其實未經官方拆分驗證（官方未公布一科內選擇/申論的時間配分）。

## 未定論／待校準（Open Questions）

1. 紅色提示是否足夠？是否該提供可選的「嚴格模式」（1.2 分/題到點即強制交卷）？
2. 混合式科目選擇題的理想時間，是否該用更精細的官方配分推估，而非統一 1.2 係數？
3. **1.2 係數可用真實作答數據重新校準**——平台會把每次作答耗時記入歷史，累積後可回看「實際選擇題作答速度」修正係數。歡迎貢獻者基於真實數據提出。
